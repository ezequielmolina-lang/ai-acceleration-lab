El Brief de Delegacion

En 2026, la habilidad clave ya no es escribir buenos prompts. Es saber que delegar a la IA, como juzgar sus resultados, y como integrarla en tu flujo de trabajo. Como dice Ethan Mollick: dirigir IA en tareas complejas se parece mas a escribir un documento de requerimientos que a escribir un prompt.

El Brief de Delegacion es tu herramienta para eso: 5 elementos que transforman un pedido vago en una instruccion precisa. Haz clic en cada tarjeta para ver un ejemplo real del Banco Mundial.

1. Tarea

Que quieres que haga la IA? Un verbo de accion claro.

Clic para ver ejemplo
Ejemplo BM

"Redacta un Informe de Regreso a la Oficina (BTOR) para mi mision de supervision del Proyecto de Caminos Rurales de Bolivia."

Clic para volver

2. Contexto

Antecedentes, audiencia y proposito del producto.

Clic para ver ejemplo
Ejemplo BM

"Fue una mision de 3 dias en La Paz. La audiencia es el Director de Pais y el equipo de GP Transporte. El proyecto esta en etapa de revision de medio termino."

Clic para volver

3. Insumos

Material en bruto: notas, datos, documentos que proporcionas.

Clic para ver ejemplo
Ejemplo BM

"Aqui estan mis notas crudas de 6 reuniones, el Aide-Memoire anterior y el ultimo ISR con sus calificaciones."

Clic para volver

4. Formato

Estructura, extension, tono y estilo del producto.

Clic para ver ejemplo
Ejemplo BM

"Usa el formato estandar de BTOR con secciones: Proposito, Reuniones Clave, Hallazgos, Tabla de Acciones Acordadas (Accion | Responsable | Plazo), Proximos Pasos. Maximo 3 paginas."

Clic para volver

5. Listo

Como sabras que el resultado es bueno? Criterios de calidad.

Clic para ver ejemplo
Ejemplo BM

"Listo cuando: todas las acciones acordadas tengan responsables y fechas, los hallazgos coincidan con mis notas, el tono sea apropiado para revision del CD, y no aparezcan datos inventados."

Clic para volver

Lo que puedes hacer con Claude Pro (gratis por 1 mes)

Aprovecha al maximo tu acceso -- estas funciones no estan disponibles en Mai

Crear un Project

  • 1. Ve a Claude.ai y haz clic en "Projects" en la barra lateral
  • 2. Crea un nuevo proyecto (ej. "LC6 Portafolio Bolivia")
  • 3. Agrega instrucciones personalizadas -- pega tu brief de delegacion o plantilla de skill aqui
  • 4. Sube documentos de referencia publicos (PADs, informes publicados)
  • 5. Cada conversacion en ese proyecto inicia con tu contexto precargado

Construir Artifacts

  • -- Dashboards: Pide a Claude que cree un Dashboard HTML interactivo para el estado de tu portafolio
  • -- Rastreadores: Construye herramientas de seguimiento de proyectos con filtros y graficos
  • -- Analizadores: Crea herramientas que procesen datos de ISR o tendencias de desembolsos
  • -- Plantillas: Genera plantillas de documentos reutilizables para Aide-Memoires, BTORs, notas de briefing
